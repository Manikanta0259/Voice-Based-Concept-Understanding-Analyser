# 7. Project Documentation

## Purpose
A place to collect user docs, developer notes, API references, and support materials for VBCUA.

## Recommended sections
- Getting started (prereqs, install, run)
  - Example commands:
    - python -m venv .venv
    - source .venv/bin/activate
    - pip install -r source/requirements.txt
    - streamlit run source/app.py

- Module reference (short descriptions + entry points)
  - `source/app.py` — UI orchestration
  - `source/audio_utils.py` — audio load/analysis/plotting
  - `source/speech_to_text.py` — ASR wrapper
  - `source/semantic_eval.py` — similarity & coverage
  - `source/scoring_engine.py` — scoring & coaching text
  - `source/report_generator.py` — PDF generation
  - `source/database.py` — DB schema & helpers

- Deployment notes
  - Required environment variables for ASR (if any)
  - ffmpeg requirements (static_ffmpeg fallback exists in `audio_utils.py`)
  - Persistent volumes for `uploads/` and `reports/`

- Contribution guide
  - Branch, commit, PR conventions
  - Testing & linting steps

- Troubleshooting & FAQ
  - Common runtime errors and fixes (missing ffmpeg, ASR model errors)
  - Where to find generated reports and waveforms
